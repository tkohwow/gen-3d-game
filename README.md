# Gen Particle Milestone Lab

白板の「Gen 2gen #3」メモを元にした、ブラウザで動く3Dゲームプロトタイプです。粒子を集めて中央の作品を成長させ、4つのマイルストーンを順に解放します。

## コンセプト

- パーティクル生成と操作をゲームの中心にする
- 2Dのポインタ操作から3Dの作品生成へつなげる
- 「最終作品を作るワークフロー」を、スコアではなくマイルストーンとして見せる
- 白板写真は入れず、3D空間内に生成した企画ボードとして配置する

## 操作

- ポインタを動かしてエミッタ位置を調整
- クリックまたはタッチ長押しで粒子を集める
- `1` / `2` / `3` または下部ボタンでモード切替
- `Space` で現在位置のエミッタを発動
- `Enter` でラン開始またはリトライ
- `R` または下部のリセットボタンでリセット

## ルール

- 1ランは60秒
- 60 Sparkで最終作品が完成
- 完成時の残り時間で`S`から`C`までのランクを判定
- 時間切れでも進捗に応じて`D`または`E`を表示
- 最高ランクはブラウザのローカルストレージに保存

## モード

- `吸`: 安定してSparkを集める
- `回`: Chainを伸ばしてSpark効率を上げる
- `作`: Chainを作品へ変換してボーナスSparkを得る

## Flow / Catalyst

- `吸 -> 回 -> 作`の順で粒子を集めるとFlowが進む
- FlowがつながるとSparkボーナスが入る
- ときどき出るCatalystを対応モードで集めると、Spark、Chain、残り時間にボーナスが入る

## 開発

```bash
pnpm install
pnpm dev
```

## ビルド

```bash
pnpm build
```
